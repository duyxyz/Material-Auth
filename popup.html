How to use?

Introduction

Welcome to NovaOTP, a lightweight and secure two-factor authentication (2FA) manager built directly into your browser.

Manual Entry

Open Settings → Manual Entry. Enter Account Name, Issuer, and Secret Key (Base32).

Google Import

Export from Google Authenticator, scan QR to get the URL, and paste into Google Import tab.

Management

• Drag & Drop: Use the handle on each card to reorder.
• Swipe: Right click and drag an item to Edit (left) or Delete (right).

GitHub Backup

Sync securely to your GitHub Gist. Data is encrypted with your Backup Password before upload.
